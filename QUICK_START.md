# 🚀 快速开始

## 完整爬取 5000+ 条评论

```bash
npm run scrape:full
```

## 详细信息

### ✅ 已做的改进

1. **轮数翻倍** - 5000+ 轮完整爬取（原来 5 轮）
2. **激进滚动** - 1000+ 条时每 50 轮用"超激进"策略
3. **超长等待** - 1500+ 条每 25 轮等待 90-120 秒
4. **智能退避** - 根据评论数自动调整策略

### 📊 性能目标

| 评论数 | 预计时间 |
|------|---------|
| 1000 | 15-30 分钟 |
| 2000 | 30-60 分钟 |
| 3000 | 1-1.5 小时 |
| 5000+ | 1.5-3 小时 |

### 🔧 自定义选项

```bash
# 指定店铺
```
